# Experiment Card 模板

实验不是「跑一下看看」。开跑前填写；存 `lab/code/experiments/<id>.md` 并登记 `lab/research/experiment-ledger.yaml`。

```markdown
# Experiment Card

## Question
What claim does this test?

## Hypothesis

## Code commit

## Config

## Data split

## Baseline / comparison

## Expected runtime

## Success metric

## Failure signals
- OOM
- NaN
- metric stall
- missing checkpoint
- data mismatch

## Artifact paths
```

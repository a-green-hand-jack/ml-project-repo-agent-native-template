# human/briefs/active/ — 进行中的任务 brief

human 下达的任务 brief 放这里。一份 brief 是任务的**权威定义**：agent 以此为准工作，不以聊天里的临时说法为准。

- 命名：`<YYYYMMDD>-<slug>.md`。
- 完成后移到 `../completed/`，并在 `memory/current-status.md` 留指针。

---

## Brief 模板（骨架）

### Objective
（一句话：要达成什么。）

### Success criteria
（可验证的完成标准，逐条列。做完能对着勾。）

### Scope
- **Allowed paths**：允许改动的路径。
- **Forbidden paths**：禁止改动的路径（agent 必须遵守）。

### Verification
（如何验证达成：命令、测试、要产出的产物路径。）

### Links
（相关 plan / issue / 实验 / 前置决策 / `lab/` 产物路径。）

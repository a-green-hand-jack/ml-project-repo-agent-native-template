# human/inbox/ — 未整理输入

零散输入的**临时落脚点**：一时看到的链接、聊天里蹦出的想法、待分类的片段、别处贴来的笔记。先扔这里，别让它们卡在对话里丢掉。

## 用法

- 随手记：一个文件一条，或一个文件多条，命名随意（建议带日期）。
- 不要求结构——`inbox/` 的价值就是低门槛。

## 定期清空

`inbox/` 是缓冲区，不是归档。定期把内容分类搬到正式位置，然后清空：

| 内容 | 去处 |
| --- | --- |
| 要做的任务 | `human/briefs/active/` |
| 值得固化的判断/取舍 | `human/decisions/`（ADR） |
| 实验事实 / 数据 | `lab/research/`、`lab/artifacts/` |
| 值得采用的 CC 工作流 | `human/reviews/recipes/` → `memory/current-practices.md` |
| 对外表述素材 | `deliverables/` |

留在 `inbox/` 太久的东西，要么升级到正式位置，要么丢弃。

## 说明

- **deprecated_at**：弃用日期（`YYYY-MM-DD`）。
- **reason**：为什么失效——模型行为变化、更好的替代、被证明有害等。
- **replaced_by**：替代 recipe 的 id（若有），指向 `current-practices.md` 或 `lab/recipes/claude-code/`。
- 弃用不等于删除：不要移除历史行，只标记；这样未来 agent 不会重新提议同一个坏做法。
